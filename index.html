🛡️ What is PhishShield?

PhishShield is an AI-powered phishing detection system that analyzes URLs in real-time to identify potential phishing threats.

🧠 ML Models Used

  • Random Forest (95.71% accuracy)
  • XGBoost (95.49% accuracy)
  • SVM (94.36% accuracy)
  • CNN (92.78% accuracy)
  • Ensemble (Combined predictions)
